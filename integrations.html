Designed For Classrooms, Not Boardrooms

LMS Integrators

If you’re familiar with leading Learning Management Systems (LMS), you may already be using BigBlueButton.

As A Synchronous Learning Tool, Our Virtual Classroom Software Is Natively Integrated Within Several LMS Platforms, And Available As A Plugin For Others.

Here are some of the most widely used BigBlueButton integrations:

  • Atutor

    The BigBlueButton integration for Atutor allows users to add live audio and video meetings to ATutor courses.

    Learn More

  • Canvas

    Canvas has integrated BigBlueButton as Canvas Conferences. Instructors can easily create conferences and invite students, and students can use Canvas Conferences from within student groups.

    Learn More

  • Chamilo

    Chamilo Learning Management System (Chamilo LMS) is a piece of software that allows users to create a virtual campus for the provision of online or semi-online training. It is distributed under the GNU/GPLv3+ license and its development process are public.

    Learn More

  • Drupal (updated)

    Drupal is an open-source content management platform powering millions of websites. This module implements the BigBlueButton API, allowing moderators to create new meetings and letting users’ meetings based on the role they have.

    Learn More

  • Docebo

    This integration enables use of BigBlueButton from within Fedena, which is an open-source school management system.

    Learn More

  • Fedena

    This integration enables use of BigBlueButton from within Fedena, which is an open-source school management system.

    Learn More

  • Foswiki

    BigBlueButton integration for Foswiki allows users to create, edit and manage meetings within the Foswiki interface.

    Learn More

  • IServ

    The IServ school server has integrated BigBlueButton as a module called “Videokonferenzen,” which allows the IServ users to hold meetings within the IServ interface.

    Learn More

  • Jenzabar

    Jenzabar collaborates with clients to make higher education more engaging and effective. Jenzabar software and services are designed to drive higher performance in every department at educational institutions.

    Learn More

  • LTI

    BigBlueButton can accept incoming LTI launch requests from a tool consumer, which is simply the IMS term for any platform that can make a LTI request to an external tool (such as BigBlueButton). Platforms include Desire2Learn, BlackBoard, and Pearson Learning Studio, etc. See IMS Interoperability Conformance Certification Status for a full list of LTI compliant platforms.

    Learn More

  • Moodle

    The Moodle activity module, created and maintained by Blindside Networks, allows users to easily create, manage and join BigBlueButton meetings from within Moodle 1.9, 2.x and 3.x. The integration is merging into the core for Moodle 4.x.

    Learn More

  • Nextcloud

    Nextcloud offers the industry-leading, on-premises content collaboration platform.

    Learn More

  • RedMine

    BigBlueButton integration for RedMine allows users to easily create, edit and manage meetings within the RedMine interface.

    Learn More

  • OpenOlat

    OpenOlat is a web-based e-learning platform for teaching, learning, assessment and communication. Within OpenOlat, BigBlueButton provides load-balancing for multiple servers and a user-centric design that allows teachers to set up online meetings with their classes.

    Learn More

  • Sakai

    Sakai has a deep integration of BigBlueButton via the Sakai Meeting tool, enabling users to easily create, manage and join BigBlueButton meetings from a Sakai worksite.

    Learn More

  • Schoology

    BigBlueButton integrates an LTI with a BigBlueButton server, allowing Schoology users to easily create, manage and join BigBlueButton meetings.

    Learn More

  • Tiki Wiki CMS

    BigBlueButton integration for Tiki users is easy. To learn more about creating, editing and managing meetings within the Tiki interface.

    Learn More

  • WordPress

    The BigBlueButton WordPress plug-in allows administrators to create and manage meeting rooms within the WordPress interface. The plugin is easily installable from within WordPress itself.

    Learn More
